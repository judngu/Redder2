# Redder

Redder is a Reddit clone done with a Ember CLI frontend and a Rails API backend.

## Installation

Backend
Move into the backend directory
```
cd backend
bundle install
rake db:setup
```

Frontend
Before you get started, make sure you have Node.js and Bower installed.
```
cd frontend
npm install
bower install
```
## Usage
```
Rake Run 
```
